本書で使用する Cygwin は下記からダウンロードすることが出来ます．

http://cygwin.com/install.html

お使いの Windows によってダウンロードするファイルが異なります．
「コントロールパネル」内の「システムとセキュリティ」内の「シス
テム」の表示ウィンドウを開いて「システムの種類」を確認します．
32ビットと書かれている場合は「x86」を64ビットの場合は「x64」を
ダウンロードします．

32ビットの場合：
Installing and Updating Cygwin for 32-bit versions of Windows
http://cygwin.com/setup-x86.exe

64ビットの場合：
Installing and Updating Cygwin for 64-bit versions of Windows
http://cygwin.com/setup-x86_64.exe

※インストール時に下記のパッケージの選択が必要です．

	gcc-core	3.4.4-999 または 4.8.3-3
	make 		3.82.90-1 または 4.0-2

標準のままインストールすると必要なソフトがインストールされませ
んので，必ず本書 p.44〜p.48をお読みください．
